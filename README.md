# 南京美食探店记录
**本文所有的店，那一定是我去过的。有些很著名也很好的馆子，我是知道的，但是我还没去，这个等一等，会更新的**  
   
作为一个研究生在南京读书的南京本地吃货，不知道南京哪里有好吃的，是一件非常可耻的事情。  
因此，本人非常热衷于探店发掘美食，更热衷于将自己的探店感受系统的推荐给诸位朋友们。希望无论是来南京旅游、学习、工作的朋友，都能找到自己喜爱的小馆子。  
**毕竟，唯有美食和美景不可辜负**  
  
***当然，所有探店点评，仅来自于我这个没见过世面的非专业小土鳖的个人评价，一家店的菜也很难全部涵盖，也受个人饮食风格影响巨大，仅供参考***

## 探店宗旨
我出去吃饭宗旨很简单，就是**物有所值**，20块钱有20块钱的标准，200有200的要求。  
正所谓一分钱一分货，花了200块钱点了菜，这钱是花在了食材上，还是技法刀工上，还是环境服务上，自己心里得门清，得让我觉得这钱花的值。  
因此本目录收录的餐厅，会偏向于两种：  
1. 口味中规中矩，但是物美价廉，性价比极高的小馆子
2. 口味优秀，价格小贵，但是对得起价钱的馆子  
  
**那些动辄人均上几百上千的高级餐馆，额，是我的问题，我穷，我吃不起，您就别往下看了**

同时我个人会有一些饮食偏好：  
1. **不在意服务和环境**   
好的服务和环境我觉得不重要，好吃才是核心，因此在我这不会加分（~~因为我穷~~）
2. **坚定的肉食动物**  
我出门点菜，基本不点蔬菜，一门心思吃肉，越多越好，武德充沛（因为我觉得我在家也能做蔬菜）
3. **因地制宜**  
南京做的最好的，肯定是淮扬菜，说实话，我到现在在南京就没吃到正宗的东北菜。这也正常，真想吃正宗的，还得到原产地去吃，这也是一大限制  
同时，由于本人住在四牌楼（行，暴露学校了），肯定周边的店会探的多一些，有些太远的，河西，江宁，仙林，那就只能随缘了
4. **对日料存在偏见**  
本科的时候非常沉迷日料，把六道口附近的所有日料店都吃了一遍。现在觉得，日料，在同等价位的情况下，相比我国八大菜系各地美食，真的性价比不高。现在除非有人喊我，我是不会主动的去吃日料的。因为日料的评价普遍也不会太高
5. **不会收录太多小吃**  
这里的小吃指的是那种简餐，比如黄焖鸡，盖浇饭，肠粉。因为太多了，这个工程量太大，而且口味基本都差不多，不好不坏。除非特别突出，值得一试，我就不收录在本文中了


## 格式规范
之后每家店会按照菜系大类进行划分，每家店按照以下属性进行介绍和评价：
* 餐馆名称
* 地址
* 菜系种类
* 菜品和评价
* 环境与服务 
* 人均价格 
* 推荐指数 
* 备注

其中推荐指数只是我个人的评价，并不具有泛用性

## 八大菜系+各地特色
### 淮扬菜
#### **— 传湌家小馆 —** 
* 地址：汉中路大锏银巷66—2（慈悲社店）
* 菜系种类：淮扬菜
* 菜品和评价：  
必点三道菜：
  - 鱼头泡油条 汤汁醇厚，吃不惯鱼的朋友可能会觉得有点腥
  - 花雕鸡 酒香浓郁，汤汁醇厚，肉质紧嫩
  - 肚肺汤 咸香开胃
* 环境与服务： 环境还行，服务也还行
* 人均价格： 俩人得120—130，小贵，但是值！
* 推荐指数： ★★★★★ **非常推荐 我的宝藏馆子list**
* 备注： 别天天往南京大排档跑了，中央厨房能有这个好吃？

#### **— 家宝风味馆 —** 
* 地址：秦淮区大香炉23号102号（就这一家）
* 菜系种类：淮扬菜
* 菜品和评价：  
性价比极高的家常菜馆子，而且口味相当不错，有滋有味
  - 黑鱼两吃 鱼片滑嫩还带点脆，鱼汤奶白鲜美鱼味足，必点！
  - 美极虾 肉质紧弹，外壳酥脆，风味十足，必点！
其他的随便点，什么小炒肉，大肠，油渣青菜，都好吃下饭
* 环境与服务： 就一个楼下的苍蝇馆子，老板老板娘夫妻店，人很热情。菜好吃就完了，你要啥服务，自己动手丰衣足食
* 人均价格： 俩人四菜一汤就70，这性价比你上哪找？
* 推荐指数： ★★★★★ **非常推荐 我的宝藏馆子list**
* 备注： 他家店铺小，而且菜都是现做，比如想吃排骨或者里脊，得等半个小时，所以最好得提前预定

#### **— 人全到饭店 —** 
* 地址：好多家，自己找
* 菜系种类：淮扬菜
* 菜品和评价：  
性价比高的家常菜馆，经济实惠，谈不上非常好吃，但是真的不贵，路过可以一试
* 环境与服务： 不同店大小不一样，反正就家常菜馆
* 人均价格： 俩人三个大肉菜都70不到，这性价比你上哪找？
* 推荐指数： ★★★ **一般推荐 性价比高的家常馆子**
* 备注： 别天天往南京大排档跑了，整点经济实惠的馆子吃吃不香么？

### 粤菜
#### **— 香港名发茶冰厅 —** 
* 地址：秦淮区中山南路89号（应该有分店）
* 菜系种类：粤菜 主打糖水
* 菜品和评价：  
我最喜欢的茶餐厅，没有之一（也有些历史原因）  
他家的饭，中规中矩，海南鸡饭和豉油鸡挺好吃，叉烧不如陈光记
但是甜点多啊，漏奶华，芝多士，菠萝包，红豆冰沙俩人都不一定能吃完，冰奶茶那是真的用奶茶结成的冰做的，不是奶茶+冰块    
当然，还有她最喜欢喝的咸柠七
* 环境与服务： 环境还行，服务也还行，就是容易人多排队
* 人均价格： 只吃份海南鸡饭也就30，但是你来这必然不是吃饭的，得人均80-90
* 推荐指数： ★★★★★ **非常推荐 我的宝藏馆子list**
* 备注： 我一般只会带女朋友去这家店，别人我从不轻易在这约饭

#### **— 澳门陈光记烧味饭店 —** 
* 地址：玄武区长江花园路长江花园11号
* 菜系种类：粤菜 主打烧腊
* 菜品和评价：  
他家主打烧腊，由于我没去过广州，所以不知道正宗的烧腊应该是啥样的，但是我觉得他们家的很好吃，不管肥叉瘦叉烧鸭，口味都不错  
但是他们家的甜点就比较一般，真想吃糖水的，请去名发  
* 环境与服务： 店面不大，桌子不大，有免费茶水，环境服务不精致也不差
* 人均价格： 只吃份叉烧饭也就30-40，点一些别的小吃就贵一些
* 推荐指数： ★★★★ **比较推荐 值得一试的烧腊馆子**
* 备注： 

### 川菜
#### **— 市井川菜 —** 
* 地址：玄武区沙塘园7号-3号（应该有分店）
* 菜系种类：川菜
* 菜品和评价：  
川菜江湖菜口味非常正的馆子，鲜香麻辣，口味经过我的四川好朋友亲自认证（但有一位吐槽份量要比四川的少）  
我们吃过的几个菜：剁椒皮蛋，麻婆豆腐（这个口味真不错），蒜泥白肉，花椒兔，都好吃
* 环境与服务： 环境还行，服务也还行
* 人均价格： 人均70-80的家常菜馆子
* 推荐指数： ★★★★★ **非常推荐 我的宝藏馆子list**
* 备注： 不建议下午要干活的时候去吃，菜及其下饭，不小心碳水就吃多了，下午贼困

## 探店宗旨
我出去吃饭宗旨很简单，就是**物有所值**，20块钱有20块钱的标准，200有200的要求。  
正所谓一分钱一分货，花了200块钱点了菜，这钱是花在了食材上，还是技法刀工上，还是环境服务上，自己心里得门清，得让我觉得这钱花的值。  
因此本目录收录的餐厅，会偏向于两种：  
1. 口味中规中矩，但是物美价廉，性价比极高的小馆子
2. 口味优秀，价格小贵，但是对得起价钱的馆子  
  
**那些动辄人均上几百上千的高级餐馆，额，是我的问题，我穷，我吃不起，您就别往下看了**

同时我个人会有一些饮食偏好：  
1. **不在意服务和环境**   
好的服务和环境我觉得不重要，好吃才是核心，因此在我这不会加分（~~因为我穷~~）  
所以，我基本上不喜欢网红店，因为大部分网红店的价格都收在了环境和服务上，大概率有坑
2. **坚定的肉食动物**  
我出门点菜，基本不点蔬菜，一门心思吃肉，越多越好，武德充沛（因为我觉得我在家也能做蔬菜）  
所以在不少店，特别是小吃店，好吃的肉类在我这是一个大的加分项  
3. **因地制宜**  
南京做的最好的，肯定是淮扬菜，说实话，我到现在在南京就没吃到正宗的东北菜。这也正常，真想吃正宗的，还得到原产地去吃，这是一大限制  
同时，由于本人住在四牌楼（行，暴露学校了），肯定周边的店会探的多一些，有些太远的，河西，江宁，仙林，那就只能随缘了
4. **对日料存在偏见**  
本科的时候非常沉迷日料，把六道口附近的所有日料店都吃了一遍。现在觉得，日料在同等价位的情况下，相比我国八大菜系各地美食，真的性价比不高。现在除非有人喊我，我是不会主动的去吃日料的。因此对日料的评价普遍也不会太高
5. **不会收录太多小吃**  
这里的小吃指的是那种简餐，比如黄焖鸡，盖浇饭，肠粉。因为太多了，这个工程量太大，而且口味基本都差不多，不好不坏。除非特别突出，值得一试，我就不收录在本文中了


## 格式规范
之后每家店会按照菜系大类进行划分，每家店按照以下属性进行介绍和评价：
* 餐馆名称
* 地址
* 菜系种类
* 菜品和评价
* 环境与服务 
* 人均价格 
* 推荐指数 
* 备注

其中推荐指数只是我个人的评价，并不具有泛用性
### 星级大致含义
| ★★★★★ | 非常推荐 | **我吃过的南京最好的美食**（可能有回忆加成） |
| ★★★★☆ | 比较推荐 | **我请好兄弟吃饭的餐馆选择表** |
| ★★★★ | 比较推荐 | **具有自己特色/独有菜品的馆子**，但只有一两道菜吸引人 |
| ★★★ | 一般推荐 | 中规中矩，不好不坏，可以试试，也可以日常吃 |
| ★★ | 个人不推荐 | 我个人不推荐的店，但是对于别人或许可以去试试看 |
| ★ | 不推荐 | 排雷，什么垃圾玩意，根本不值得一去 |

#### **— 家宝风味馆 —** 
* 地址：秦淮区大香炉23号102号（就这一家）
* 菜系种类：淮扬菜
* 菜品和评价：  
性价比极高的家常菜馆子，而且口味相当不错，有滋有味  
随便点，什么小炒肉，大肠，油渣青菜，都好吃下饭
  - 黑鱼两吃 鱼片滑嫩还带点脆，鱼汤奶白鲜美鱼味足，必点！
  - 美极虾 肉质紧弹，外壳酥脆，风味十足，必点！ 
* 环境与服务： 就一个楼下的苍蝇馆子，老板老板娘夫妻店，人很热情。菜好吃就完了，你要啥服务，自己动手丰衣足食

#### **— 奇食大不同菜馆 —** 
* 地址：秦淮区仙鹤街35号
* 菜系种类：淮扬菜（我觉得算改良川菜）
* 菜品和评价：  
这家店在我收藏夹里躺很久了，昨天去吃，确实有点东西  
他家招牌菜三板斧：  
  - 红烧牛尾 私以为最好吃的红烧牛尾是北京鲁菜正统鸿宾楼的，这家店有七份相似；牛肉软烂入味，牛肉味十足，红烧口正，汤汁底也鲜亮；应该是煮好了一大锅，然后有人吃就从中舀一盘；就是150一盘，有点小贵，但对得起这个价钱  
  - 剁椒全腰 不是腰花，是带着腰骚的全腰，然而吃着一点骚味都没有；口感软烂适中带点脆；剁椒很香，看似红油很多，实则稍微带点辣口，必点！
  - 宫保双臭 一开始我以为是肥肠+臭豆腐，额，不是，单纯是臭豆腐；简单理解就是宫保鸡丁+过油炸的臭豆腐丁；因为过油炸了，所以臭豆腐不臭，是干脆酥香的口感，配到宫保鸡丁里面，非常合适；就是这道菜他们家甜口大，不喜欢吃甜的朋友可能不适应    
* 环境与服务： 服务环境都还行，上菜速度很快
* 人均价格： 可以吃到人均120，但基本就80-90左右
* 推荐指数： ★★★★☆ **比较推荐 很有特色的淮扬菜改良川菜馆子**
* 备注： 他家店铺小，人很多，翻台非常快，要么早去早吃，要么提前预定；而且他家做菜偏甜口，不喜欢吃甜的朋友不一定习惯（我个人觉得是能接受的）

#### **— 大风歌食府 —** 
* 地址：玄武区龙蟠中路29号 （离学校很近，有分店）
* 菜系种类：徐州菜
* 菜品和评价：  
徐州菜原本是鲁菜的一个分支，现在算是介于淮扬菜和鲁菜之间，还受豫菜影响，算是博采众长  
这家店靠近军区，讲的就是份量大实惠，真的有种在北方吃饭的快乐，盘子比脸盆都大。就冲这一点，我就可以给到四星  
他们家下料足，吃起来有滋有味，口味都很家常，谈不上非常的惊艳，但绝对下饭好吃  
我平时这个不怎么吃辣的人，在他们家，能吃到市井川菜辣的爽快感，疲惫一扫而光  
  - 大盘鸡就是在家自己烧鸡的味道，口味透，满满一盘
  - 沛县面皮 凉皮，辣油香，醋也香，非常的开胃
  - 胡辣汤 这个胡辣汤有点普通（没我在北京金狮麟吃的好），但是满满一盆，下料狠，配个烤馍真的绝配，我最后连喝好几碗非常过瘾  
  - 蒜蓉时蔬 平时我是不提蔬菜的，但是这家店深得我心。用蒜量极大，和我做菜一个风格，我很喜欢。最后一大盘生菜基本给我吃完了
  - 孜然羊肉 说是招牌，份量少了一些，给的卷饼差点意思，冷的还干硬没嚼劲。羊肉本身没啥问题，而且还有些嚼劲，越嚼越香，还是不错的  
* 环境与服务： 有门面，环境服务都还行
* 人均价格： 80-90
* 推荐指数： ★★★★☆ **比较推荐 性价比极高口味不错的徐州家常菜馆子**

#### **— 吉事利茶餐厅 —** 
* 地址：玄武区峨嵋路北极西村9号103室(高楼门4号、近奶酪时光)
* 菜系种类：粤菜
* 菜品和评价：  
粤菜家常菜馆子，点了几个招牌菜：

#### **— 苏式现浇面馆 —**

所以从某种程度来说，这家苏式面，它不正宗，因为它糖放的不够多。但也因为糖不多，我觉得吃着还挺好，整个大肉再加一份虾仁三鲜，价格不贵好吃实惠    
缺点就是虾仁三鲜的份量有点小，大肉还是好吃的

#### **— 鸡鸣寺百味斋——面馆 —** 
* 地址：鸡鸣寺登到顶，上香的那边（这是面馆，点菜的在后门口）
* 菜系种类：面馆
* 菜品和评价：  
鸡鸣寺的素面早有耳闻，也是趁着今年中秋，剃了光头，中午去吃顿素面吧  
百味斋是有俩门面的，鸡鸣寺里面那个是面条+点心，后门出来的那个是用来点菜的菜馆，菜馆之后有机会去试试  
这里的素食是忌肉蛋奶的，我基本把点心都吃了一遍：  
  - 素面 额，讲真一般般，没啥惊艳的，油也有点大，吃着觉得腻（不过碗底有朵樱花）  
  - 素煎鸡蛋 强烈推荐，一定要试试 蛋白我吃不出来是啥做的，蛋黄好像是豌豆黄调的，真的和煎鸡蛋口感一摸一样，蛋白细腻，蛋黄有沙沙的口感，不告诉你是素鸡蛋根本吃不出来，绝了  
  - 烧卖 这玩意想好吃，就得有油，所以素烧卖，有点干  
  - 如意糕 就一方糕，不好吃，噎人，别买  
  - 素烧鹅 就豆皮，甜口，油重，份量不少，单吃不好吃，得放面里  
* 环境与服务： 服务环境还行，排队人多，容易没位置
* 人均价格： 人均40-50
* 推荐指数： ★★★ **一般推荐 可以尝鲜，除非是吃素吃惯的，那得相信，基因告诉你肉比菜好吃**
* 备注：咋说呢，由于是纯素食，又要保证口感，那用的油就要多了，所以吃下来会觉得有点腻，这点得提醒一下  

#### **— 刘长兴 —** 
* 地址：有很多分店 最近的在鼓楼
* 菜系种类：南京小吃集锦
* 菜品和评价：  
众所周知，百年老店十个里面有八个都不太行，刘长兴属于无功无过的那种，能吃，但放在现在已经不算突出了  
他家种类多，什么汤包，蒸饺，馄饨，面条，炒饭，点心，乃至小炒都有，价格也便宜  
我个人很喜欢他们家的鱼汤馄饨，8块钱一碗小馄饨，里面有鱼丸，汤底也能喝出鱼味  
至于他家的汤包，额，我觉得无功无过，要是没尝过灌汤包，来他们家吃不算踩雷    
汤包这玩意现在很矛盾，毕竟是用皮冻+荤油做的汤汁，想好吃就得用肥肉多的馅，荤油还得多，但是现代大家口味没那么重，这么搞容易腻。但是汤和馅淡了鲜味口感出不来（最直观的就是蘸醋没有解腻的感觉了），当然时代在变化，我们除了吃的好吃，也得健康  
菠萝酥也挺好吃，2.6一个，馅应该是自己调的，才烤出来冒着热气捏在手里表皮酥脆（讲真，现在说在南京买点心，好多店还不如去泸溪河呢，说的就是你绿柳居）    
* 环境与服务： 服务环境还行
* 人均价格： 人均20-30
* 推荐指数： ★★★ **一般推荐 分店很多路过可以一试**

### 各种饭（抓饭/盖浇饭/黄焖鸡）

* 地址：玄武区吉兆营吉兆花园4号楼101室
* 菜系种类：抓饭
* 菜品和评价：  
第一次吃到他们家的羊肉，有被惊艳到，羊肉质量很不错，肉质软嫩入味，没有任何不好的膻骚味，甚至有点奶香  
而且调味简单，没有上什么孜然烧烤料，清水羊肉做出来应该就是这个味道  
吃完之后，我就非常想和他们店里讨一个方子回家自己煮羊肉，这是我心中羊肉白煮+简单调味去膻腥，应该做成的样子

至于其他的，什么奶茶，小菜，都是添头，不要被大众点评上那些奇怪的推荐给蒙了

* 推荐指数： ★★★★ **比较推荐 羊肉做的很不错的一家手抓饭**

### 其他玩意（米线/凉粉/锅贴/肠粉）
